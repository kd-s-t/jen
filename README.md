## Quick Start

**Prerequisites:**
- Node.js >=20.0.0
- npm >=10.0.0

**Setup and Run:**
```bash
npm install
npm run dev
```

The development server runs on port **3026** at `http://localhost:3026`.

**Live Site:**
Visit [https://jen-orcin.vercel.app/](https://jen-orcin.vercel.app/)
